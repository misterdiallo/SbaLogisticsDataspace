Logistics Connector

SBA Logisctics Dataspace Connector provides Logistics Connector inter-organizational data exchange. It implements the International Data Spaces standard (IDS) as well as relevant protocols associated with GAIA-X. The connector is designed in an extensible way in order to support alternative protocols and integrate in various ecosystems.

Logistics Connector Data Dashboard

This Logistics Connector Data Dashboard is a UI for the Logistics Connector Management.

Example use cases, that you can try out with this application, are:

  • View the asset catalog available to you in the Logistics Dataspace using the Catalog Browser
  • Negotiate a contract for data sharing in the Logistics Dataspace using the Catalog Browser
  • View your existing contracts in the Contracts page
  • Transfer an asset in the Logistics Dataspace using the Contracts page
  • View which assets have been transferred in the Logistics Dataspace in the Transfer History page
  • View and create assets using the Assets page
  • View and create policies and apply these to assets in the Logistics Dataspace using the Policies page
  • Publish a new asset into the Logistics Dataspace using the Contract Definitions page
